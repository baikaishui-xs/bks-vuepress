# 知识点：排他思想

**排他思想：** 当前元素设置样式，其余的兄弟元素清除样式

```html
<button>快速</button>
<button>快速</button>
<button>快速</button>
<button>快速</button>
<button>快速</button>
<button>快速</button>
<button>快速</button>
```
```js
$("button").click(function() {
    $(this).css("background", "pink");
    $(this).siblings("button").css("background", "");
});
```
