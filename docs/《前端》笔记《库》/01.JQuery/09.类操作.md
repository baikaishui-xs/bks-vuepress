# 知识点：类操作

**特性：**
1. 原生 JS 中 className 会 覆盖原先的类名
2. JQuery 中的类操作 不会 覆盖原先的类名

## 一、添加类

**方法：** `$('元素').addClass('类名');`

```js
$("button").click(function() {
    $(this).addClass("current");
});
```

## 二、移除类

**方法：** `$('元素').removeClass('类名');`

```js
$("button").click(function() {
    $(this).removeClass("current");
});
```

## 三、切换类

**方法：** `$('元素').toggleClass('类名');`

```js
$("button").click(function() {
    $(this).toggleClass("current");
});
```
