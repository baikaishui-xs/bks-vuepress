# 属性
  | 属性   |    说明    |    类型 | 可选值                                                                                     | 默认值  |
  | :----- | :--------: | ------: | :----------------------------------------------------------------------------------------- | :-----: |
  | text   |  标签内容  |  String | -                                                                                          |    -    |
  | type   |  颜色类型  |  String | default（灰色）、primary（蓝色）、success（绿色）、warning(黄色)、error(红色)、royal(紫色) | default |
  | circle | 是否为圆角 | Boolean | false                                                                                      |    -    |

# 方法 
  | 方法   |         说明         | 参数                   |
  | :----- | :------------------: | :--------------------- |
  | @input | 输入事件。输入后触发 | 参数一：用户输入的内容 |

# 一、基础结构
  ```html
  <uni-tag text="标签" type="error" :circle="true"></uni-tag>
  ```
