# 一、公共组件目录名称需遵循 大驼峰命名法
  **如：**
  - components/DemoDemo/index.vue
  - components/DemoDemo1/index.vue

# 二、路由组件目录名称需遵循 小驼峰命名法
  **如：**
  - views/demoDemo/index.vue
  - views/demoDemo1/index.vue

# 三、组件名称的 name 配置项需遵循 大驼峰命名法
  ```js
  export default {
    name: 'Demo'
  }
  ```
