在 vue 2.x 的版本中，<template> 节点内的 DOM 结构仅支持单个根节点

  <template>
    <div>
      <h1>这是 App 根组件</h1>
      <h2>这是副标题</h2>
    </div>
  </template>

在 vue 3.x 的版本中，<template> 中支持定义多个根节点

  <template>
    <h1>这是 App 根组件</h1>
    <h2>这是副标题</h1>
  </template>
