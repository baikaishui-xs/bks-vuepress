# 知识点：WebAPI

## WebAPI
是浏览器提供的一套操作浏览器功能和页面元素的 API。由 DOM（页面文档对象模型） 和 BOM（浏览器对象模型） 组成

## API（接口）
预先定义的函数，可以轻松实现某种功能，无需访问源码和理解内部工作原理
