# 知识点：

**作用：** 简化代码，提高性能。不需要为每个子节点添加事件，也解决了访问 DOM 次数过多，导致性能降低 的问题

**原理：** 为父元素添加事件监听，通过冒泡原理影响每个子节点

**特性：** 通常搭配 target 使用

```html
<ul>
    <li>知否知否，点我应有弹框在手！</li>
    <li>知否知否，点我应有弹框在手！</li>
    <li>知否知否，点我应有弹框在手！</li>
    <li>知否知否，点我应有弹框在手！</li>
    <li>知否知否，点我应有弹框在手！</li>
</ul>
```
```js
var ul = document.querySelector('ul');
ul.addEventListener('click', function(e) {
    e.target.style.backgroundColor = 'pink';
})
```
