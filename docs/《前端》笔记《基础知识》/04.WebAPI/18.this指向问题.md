# 知识点：this指向问题

## 一、只有函数执行，才能确定 this 指向

## 二、谁调用 this 所在的函数，this 就指向谁

## 三、全局作用域、普通函数 中的 this 指向 window
```js
console.log(this);
function demon1 () {
    console.log(this);
}
demon1();
```

## 四、定时器的 this 指向 window
```js
setTimeout(function () {
    console.log(this);
})
```
