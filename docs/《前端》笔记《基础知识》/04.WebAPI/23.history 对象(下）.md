# 知识点：history 对象(下）

**作用：**操作浏览器历史记录

## 三、允许后退

```html
<a href="20-history 对象(上）.html">点击我去往首页</a>
<button>后退</button>
```
```js
var btn = document.querySelector('button');
```

**方法：** history.back()

btn.addEventListener('click', function() {

})
