# 知识点：高阶函数

以下两种语法被称为高阶函数

## 一、把函数作为参数（如：回调函数）
```js
function fn (callback) {}
fn(function() {})
```

## 二、将函数作为返回值输出
```js
function fn () {
    return function () {}
}
fn()
```

# demo
```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>Document</title>
</head>
<body>
    <script>
        // 以下两种语法被称为高阶函数

        // （1）把函数作为参数  如：回调函数
        function fn (callback) {}
        fn(function() {})

        // （2）将函数作为返回值输出
        function fn () {
            return function () {}
        }
        fn()
    </script>
</body>
</html>
```
