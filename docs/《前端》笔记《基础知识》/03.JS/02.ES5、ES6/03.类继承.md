# 知识点：类继承

**作用：** 子类继承父类的属性和方法

**语法：** class 子类名 extends 父类名 {}

**super 关键字：** 父类

```js
class Father {
    constructor(x, y) {
        this.x = x;
        this.y = y;
    }
    sum() {
        console.log(this.x + this.y);
    }
    say() {
        console.log('继承方法');
    }
}
```

## 一、继承属性

**方法：** `super`

```js
class Son extends Father {
    constructor(x, y) {
        super(x, y);
    }
}
```

## 二、继承方法

**方法：** `class 子类 extends 父类 {}`

**特性：** 子类有和父类都有该方法时，只使用子类的

```js
class Son extends Father {
    say() {
        console.log('子类有和父类都有该方法时，只使用子类的');
        super.say();  // 使用父类方法
    }
}
```

```js
var son = new Son(1, 2);
var son1 = new Son(11, 22);
son.x;  // 1
son.say();  // 继承方法
```
