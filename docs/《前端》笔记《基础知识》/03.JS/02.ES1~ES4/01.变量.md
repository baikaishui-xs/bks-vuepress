# 知识点：变量

**特性：**
1. 只声明 不赋值：undefied
2. 不声明 不赋值 直接使用：报错
3. 不声明 只赋值：全局变量

**同时声明多个变量：**
```js
var age = 18, name = 'zs', sex = 2;
```
