# 知识点：对象合并

**作用：** 将其它对象，合并到目标对象中

**特性：** 只能拷贝简单数据类型，赋值数据类型拷贝的是地址

```js
var obj = {
    id: 1,
    name: 'andy',
    msg: {
        age: 18
    }
};
var o = {};

for (var k in obj) {
    o[k] = obj[k];
}
console.log(o);
o.msg.age = 20;
console.log(obj);
```
