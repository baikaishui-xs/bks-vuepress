# 知识点：指数操作符

**作用：** 幂运算

**语法：** `**`

**特性：** 和 Math.pow 作用一样

```js
// 计算 2 的 10次方
2 ** 10;  // 1024
Math.pow(2, 10);  // 1024
```

# demo
```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>Document</title>
</head>
<body>
    <script>
        // 知识点：指数操作符
        // 作用：幂运算
        // 语法：**
        // 特性：和 Math.pow 作用一样

        // 计算 2 的 10次方
        2 ** 10;  // 1024
        Math.pow(2, 10);  // 1024
    </script>
</body>
</html>
```
