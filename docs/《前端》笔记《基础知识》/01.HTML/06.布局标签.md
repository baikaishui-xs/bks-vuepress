# 知识点：布局标签

## div

```html
<div></div>
```

**作用：** 大盒子

**特性：** 独占一行

**渲染后：**
> <div>大盒子，独占一行</div> <div>大盒子，独占一行</div>

## span

```html
<span></span>
```

**作用：** 小盒子

**特性：** 一行显示多个

**渲染后：**
> <span>小盒子，一行可以显示多个</span> <span>小盒子，一行可以显示多个</span> <span>小盒子，一行可以显示多个</span>
