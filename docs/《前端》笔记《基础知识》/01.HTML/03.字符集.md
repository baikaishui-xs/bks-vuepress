# 知识点：字符集

**作用：** 规定 HTML 文档应该使用哪种字符编码

**UTF-8：** 基本包含了全世界所有国家需要用到的字符

```html
 <!DOCTYPE html>
<html lang="zh-CN">
<head>
    <!-- charset（字符集） -->
	<meta charset="UTF-8">
	<title>Document</title>
</head>
<body>
	你好吗 
</body>
</html>
```
