# 知识点：文本格式化标签

## 一、加粗

```html
<strong>加粗</strong>
```

**渲染后：**
> <strong>加粗</strong>

## 二、倾斜

```html
<em>倾斜</em>
```

**渲染后：**
> <em>倾斜</em>

## 三、删除线

```html
<del>删除线</del>
```

**渲染后：**
> <del>删除线</del>


## 四、下划线

```html
<ins>下划线</ins>
```

**渲染后：**
> <ins>下划线</ins>
