# 知识点：三角形

**使用步骤：**
1. 定义 border
2. 定义 border-left-color

**例：**
```css
.demo {
    border: 50px solid transparent;
    border-left-color: red;
}
```

**效果：**
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>Document</title>
    <style>
        .demo {
            border: 50px solid transparent;
            border-left-color: red;
        }
    </style>
</head>
<body>
    <div class="demo"></div>
</body>
</html>
