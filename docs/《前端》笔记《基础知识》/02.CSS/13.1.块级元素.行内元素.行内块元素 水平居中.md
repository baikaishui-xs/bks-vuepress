## 一、块级元素 水平居中

**使用步骤：**
1. 为盒子指定宽度
2. 设置盒子左右的外边距都为 auto

**例：**
```css
.demo {
    width: 400px;
    margin: 0 auto;
}
```

## 二、行内元素、行内块元素 水平居中

**使用步骤：** 为父元素添加 `text-align: center`
