# 知识点：数值允许使用公式计算

**原理：** `calc 函数`

**效果：**
```css
.demo {
    width: calc(100% - 80px);
    height: 200px;
    background-color: pink;
}
```
```html
<div class="demo"></div>
```

<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>Document</title>
    <style>
        .demo {
            width: calc(100% - 80px);
            height: 200px;
            background-color: pink;
        }
    </style>
</head>
<body>
    <div class="demo"></div>
