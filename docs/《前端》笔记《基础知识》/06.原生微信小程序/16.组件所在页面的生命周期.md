说明：有时，自定义组件的行为依赖于页面状态的变化，此时就需要用到组件所在页面的生命周期。如 在组件中监听页面状态的变化

生命周期函数

  pageLifetimes: {

    （1）show
      触发条件：组件所在页面 被展示后
      
    （2）hide
      触发条件：组件所在页面 被隐藏后

    （3）resize
      触发条件：组件所在页面 尺寸变化后

  }
